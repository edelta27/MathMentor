# MathMentor

MathMentor to aplikacja mobilna pomagająca dzieciom uczyć się tabliczki mnożenia w przyjazny sposób.

## Funkcje

- tryb ćwiczeń
- tryb testu (20 przykładów)
- obsługa dwóch języków
- prosty i czytelny interfejs dla dzieci

## Technologie

- Java
- Android Studio
- Gradle
